# Archive

This folder is intended for legacy or temporary files that are not part of the current app flow.

Use this folder to keep old samples, experiment files, or duplicate code separate from the working product.

- `archive/docs-duplicate/` should hold legacy frontend duplicates from `docs/`
- `archive/legacy-html/` should hold temporary HTML files such as `code1.html` through `code9.html`
- `archive/scripts/` can hold helper tools like PDF extraction scripts once they are moved from the repo root
